# Obsidian笔记软件


# notion和Obsidian


**Notion 像一个“万能办公桌”**


你可以在上面记笔记、做清单、排任务、做表格、和别人一起改内容。


**Obsidian 像一个“只属于你自己的笔记仓库”**


更适合一个人安安静静写东西、整理想法、长期保存。


最简单的区别就是：


**Notion = 更适合“做事”**


**Obsidian = 更适合“想事”**



## 先说 Notion


Notion 的好处是：


**1. 上手比较容易**


界面比较直观，点一点就能建页面、加表格、做待办。


**2. 很适合整理工作和学习计划**


比如课程表、读书清单、项目进度、团队文档，都很方便。


**3. 很适合多人一起用**


你可以和同学、同事一起看、一起改，这点它很强。


**4. 页面看起来比较整齐好看**


很多模板直接套就行，小白会比较有安全感。


它的缺点是：


**1. 更依赖网络和平台**


虽然现在也能离线用一点，但整体感觉还是更偏“在线工具”。


**2. 容易越搞越复杂**


一开始只是记笔记，后来可能变成天天折腾页面、表格、模板，结果笔记没写多少。


**3. 更像“工具箱”**


适合管理事情，但不一定最适合深入思考。



## 再说 Obsidian


Obsidian 的好处是：


**1. 你的笔记是存在自己电脑里的**


这个很重要。意思就是：你的东西更像真正属于你自己，不太怕平台以后变动。


**2. 很适合长期记笔记**


比如读书笔记、写作素材、灵感、知识整理，用久了会很舒服。


**3. 特别适合把想法串起来**


你今天写一个概念，明天写另一个，可以互相链接，慢慢形成自己的知识网。


**4. 写字体验比较纯粹**


少一些花里胡哨，更像专心写内容。


它的缺点是：


**1. 对小白没那么友好**


刚开始会觉得：这啥？文件夹、Markdown、双链、插件……有点懵。


**2. 协作不如 Notion**


如果你想和别人一起编辑，Notion 通常更省心。


**3. 太自由了，也容易折腾**


很多人会沉迷装插件、改主题，最后又开始“研究工具”而不是“记笔记”。



## 总结


**Notion：像一个共享办公室，热闹、全能、适合一起干活。**


**Obsidian：像一个私人书房，安静、自由、适合自己沉淀。**



# Obsidian 里“知识链”到底是啥


**Obsidian 里“知识链”不是啥高深理论，本质上就是：一条笔记，能和另一条笔记形成“来龙去脉”的关系网。**


不是只有“我点过去”，还包括“谁提到了我”“我跟谁有关”“这一串东西最后连成了一张网”。Obsidian 官方就把它叫“通过链接笔记，创建知识网络”；它还有专门的 Backlinks 面板，能看到哪些笔记链接到了当前笔记，甚至还能显示没正式加链接、但提到了这个词的 “unlinked mentions”。



# “Obsidian + Claude Code + Karpathy”


Karpathy 4 月 4 日发的那个 `LLM Wiki` gist，核心意思不是“让 AI 替你变聪明”，而是：


**不要每次问 AI，都让它重新从一堆原始资料里临时翻答案；不如让 AI 持续维护一个“长期存在的、互相链接的 markdown wiki”。** 这个 wiki 会随着你加入新资料不断更新、补链接、改总结、标出冲突。Karpathy 自己甚至直接把这个模式形容成：**Obsidian 像 IDE，LLM 像程序员，wiki 像代码库。**


再翻译成人话，就是：


你有一堆资料：文章、播客笔记、会议记录、论文、网页。


以前你是：


**把资料丢给 AI → AI 当场现找 → 回你一个答案**


Karpathy 这个思路是：


**先让 AI 把这些资料整理成一套长期维护的知识库页面**


比如：

- 人物页
- 概念页
- 对比页
- 总结页
- 索引页

以后你再问问题，AI 不是每次重做一遍，而是基于这套已经整理过的 wiki 继续回答和更新。这个 wiki 会“越用越厚”，而不是“每次聊天都从零开始”。



# 那 Claude Code 在里面是干嘛的？


Claude Code 官方定位就是一个能在终端、IDE、网页里工作的 coding assistant，可以读文件、改文件、跑命令，而且改文件前会征求同意。


所以很多人把它接到 Obsidian 上，本质是因为：


**Obsidian 里的内容就是一堆 markdown 文件，AI 很好读，也很好改。**


于是 Claude Code 就像一个“自动整理笔记的助理”：

- 帮你读新资料
- 帮你生成摘要页
- 帮你在旧页面里补链接
- 帮你更新索引
- 帮你把零散聊天沉淀成正式笔记

这也是为什么这套东西听起来很唬人，但本质非常朴素：


**Obsidian 提供笔记文件和链接结构，Claude Code 负责干体力活**
